bci_essentials.bci_controller

Module for managing BCI data.

This module provides data classes for different BCI paradigms.

It includes the loading of offline data in xdf format or the live streaming of LSL data.

The loaded/streamed data is added to a buffer such that offline and online processing pipelines are identical.

Data is pre-processed (using the signal_processing module), divided into trials, and classified (using one of the classification sub-modules).

Classes
  • BciController : For processing continuous data in trials of a defined length.

199    def step(self):
200        """Runs a single BciController processing step.
201
202        See setup() for configuration of processing.
203
204        The method:
205        1. Pulls data from sources (EEG and markers).
206        2. Run a while loop to process markers as long as there are unprocessed markers.
207        3. The while loop processes the markers in the following order:
208            - First checks if the marker is a known command marker from self.marker_methods.
209            - Then checks if it's an event marker (contains commas)
210            - If neither, logs a warning about unknown marker type
211        3. If the marker is a command marker, handles it by calling __handle_command_marker().
212        4. If the marker is an event marker, handles it by calling __handle_event_marker().
213        5. If the command or event marker handling return continue_flag as True, increment the marker count and process the next marker.
214            - Note: If there is an unknown marker type, the marker count is still incremented and processing continues.
215        6. If the command or event marker handling return continue_flag as False, break out of the while loop and end the step.
216
217        Parameters
218        ----------
219        `None`
220
221        Returns
222        ------
223        `None`
224
225        """
226        # read from sources to get new data.
227        # This puts command markers in the marker_data array and
228        # event markers in the event_marker_strings array
229        self._pull_data_from_sources()
230
231        # Process markers while there are unprocessed markers
232        # REMOVE COMMENT: check if there is an available command marker, if not, break and wait for more data
233        while len(self.marker_timestamps) > self.marker_count:
234            # Get the current marker
235            current_step_marker = self.marker_data[self.marker_count]  # String
236            current_timestamp = self.marker_timestamps[self.marker_count]  # Float
237
238            # If messenger is available, send feedback for each marker received
239            if self._messenger is not None:
240                self._messenger.marker_received(current_step_marker)
241
242            # Process markers in order specified in the docstrings
243            # First check if it's a known command marker
244            if current_step_marker in self.marker_methods:
245                continue_flag = self.__handle_command_marker(current_step_marker)
246            # Then check if it's an event marker (contains commas)
247            elif "," in current_step_marker:
248                continue_flag = self.__handle_event_marker(
249                    current_step_marker, current_timestamp
250                )
251            # Otherwise, log a warning about unknown marker type
252            else:
253                # Log warning for unknown marker types
254                logger.warning("Unknown marker type received: %s", current_step_marker)
255
256            # Check if we should continue processing markers in the while loop
257            # if continue_flag is False, then break out of the while loop
258            # else, increment the marker count and process the next marker
259            if continue_flag is False:
260                break
261            else:
262                logger.info("Processed Marker: %s", current_step_marker)
263                self.marker_count += 1
264

266        """Runs BciController processing in a loop.
267
268        See setup() for configuration of processing.
269
270        Parameters
271        ----------
272        max_loops : int, *optional*
273            Maximum number of loops to run, default is `1000000`.
274
275        Returns
276        ------
277        `None`
278
279        """
280        # if offline, then all data is already loaded, only need to loop once
281        if self.online is False:
282            self.loops = max_loops - 1
283        else:
284            self.loops = 0
285
286        # Initialize the event marker buffer
287        self.event_marker_buffer = []
288        self.event_timestamp_buffer = []
289
290        # start the main loop, stops after pulling new data, max_loops times
291        while self.loops < max_loops:
292            # print out loop status
293            if self.loops % 100 == 0:
294                logger.debug(self.loops)
295
296            if self.loops == max_loops - 1:
297                logger.debug("last loop")
298
299            # read from sources and process
300            self.step()
301
302            # Wait a short period of time and then try to pull more data
303            if self.online:
304                time.sleep(0.00001)
305
306            self.loops += 1

334    def __pull_marker_data_from_source(self):
335        """Pulls marker samples from source, sanity checks and appends to buffer.
336
337        Parameters
338        ----------
339        `None`
340
341        Returns
342        -------
343        `None`
344
345        """
346
347        # if there isn't a marker source, abort
348        if self.__marker_source is None:
349            return
350
351        # read in the data
352        markers, timestamps = self.__marker_source.get_markers()
353        markers = np.array(markers)
354        timestamps = np.array(timestamps)
355
356        if markers.size == 0:
357            return
358
359        if markers.ndim != 2:
360            logger.warning("discarded invalid marker data")
361            return
362
363        # apply time correction
364        time_correction = self.__marker_source.time_correction()
365        timestamps = [timestamps[i] + time_correction for i in range(len(timestamps))]
366
367        for i, marker in enumerate(markers):
368            marker = marker[0]
369            if "Ping" in marker:
370                continue
371
372            # Add all markers to the controller
373            self.marker_data = np.append(self.marker_data, marker)
374            self.marker_timestamps = np.append(self.marker_timestamps, timestamps[i])
375
376            # Add all markers to the data tank
377            self.__data_tank.add_raw_markers(
378                np.array([marker]), np.array([timestamps[i]])
379            )

381    def __pull_eeg_data_from_source(self):
382        """Pulls eeg samples from source, sanity checks and appends to buffer.
383
384        Parameters
385        ----------
386        `None`
387
388        Returns
389        -------
390        `None`
391
392        """
393
394        # read in the data
395        eeg, timestamps = self.__eeg_source.get_samples()
396        eeg = np.array(eeg)
397        timestamps = np.array(timestamps)
398
399        if eeg.size == 0:
400            return
401
402        if eeg.ndim != 2:
403            logger.warning("discarded invalid eeg data")
404            return
405
406        # if time is in milliseconds, divide by 1000, works for sampling rates above 10Hz
407        try:
408            if self.time_units == "milliseconds":
409                timestamps = [(timestamps[i] / 1000) for i in range(len(timestamps))]
410
411        # If time units are not defined then define them
412        except Exception:
413            dif_low = -2
414            dif_high = -1
415            while timestamps[dif_high] - timestamps[dif_low] == 0:
416                dif_low -= 1
417                dif_high -= 1
418
419            if timestamps[dif_high] - timestamps[dif_low] > 0.1:
420                timestamps = [(timestamps[i] / 1000) for i in range(len(timestamps))]
421                self.time_units = "milliseconds"
422            else:
423                self.time_units = "seconds"
424
425        # apply time correction, this is essential for headsets like neurosity which have their own clock
426        time_correction = self.__eeg_source.time_correction()
427        timestamps = [timestamps[i] + time_correction for i in range(len(timestamps))]
428
429        self.__data_tank.add_raw_eeg(eeg.T, timestamps)
430
431        # Update latest EEG timestamp
432        self.latest_eeg_timestamp = timestamps[-1]

465    def __process_and_classify(self):
466        """Process the markers and classify the data.
467
468        Parameters
469        ----------
470        `None`
471
472        Returns
473        -------
474        success_string : str
475            String indicating if the processing and classification was successful.
476            Potential values are "Success", "Skip", "Wait".
477            - "Success": The processing and classification was successful.
478            - "Skip": EEG is either absent entirely or contains lost packets.
479            - "Wait": The processing is waiting for more data.
480
481        """
482
483        eeg_start_time, eeg_end_time = self.__paradigm.get_eeg_start_and_end_times(
484            self.event_marker_buffer, self.event_timestamp_buffer
485        )
486
487        # No we actually need to wait until we have all the data for these markers
488        eeg, timestamps = self.__data_tank.get_raw_eeg()
489
490        # Check if there is available EEG data
491        if len(eeg) == 0:
492            logger.warning("No EEG data available")
493            return "Skip"
494
495        # If the last timestamp is less than the end time, then we don't have the necessary EEG to process
496        if timestamps[-1] < eeg_end_time:
497            return "Wait"
498
499        # Check if EEG sampling is continuous over this time period
500        start_indices = np.where(timestamps > eeg_start_time)[0]
501        if len(start_indices) == 0:
502            logger.warning("No timestamps exceed eeg_start_time")
503            return "Skip"
504        start_index = start_indices[0]
505        end_index = np.where(timestamps < eeg_end_time)[0][-1]
506
507        time_diffs = np.diff(timestamps[start_index:end_index])
508        if np.any(time_diffs > 2 / self.fsample):
509            logger.warning("Time gaps in EEG data")
510            return "Skip"
511
512        X, y = self.__paradigm.process_markers(
513            self.event_marker_buffer,
514            self.event_timestamp_buffer,
515            eeg,
516            timestamps,
517            self.fsample,
518        )
519
520        # Add the epochs to the data tank
521        self.__data_tank.add_epochs(X, y)
522
523        # If either there are no labels OR iterative training is on, then make a prediction
524        if self.train_complete:
525            if -1 in y or self.__paradigm.iterative_training:
526                prediction = self._classifier.predict(X)
527                self.__send_prediction(prediction)
528
529        self.event_marker_buffer = []
530        self.event_timestamp_buffer = []
531
532        return "Success"

534    def __update_and_train_classifier(self):
535        """Updates the classifier if required.
536
537        Parameters
538        ----------
539        `None`
540
541        Returns
542        -------
543        continue_flag : bool
544            Flag indicating to continue the while loop in step().
545        """
546        if self.train_lock is False:
547            # Pull the epochs from the data tank and pass them to the classifier
548            X, y = self.__data_tank.get_epochs(latest=True)
549
550            # Remove epochs with label -1
551            ind_to_remove = []
552            for i, label in enumerate(y):
553                if label == -1:
554                    ind_to_remove.append(i)
555            X = np.delete(X, ind_to_remove, axis=0)
556            y = np.delete(y, ind_to_remove, axis=0)
557
558            # Check that there are epochs
559            if len(y) > 0:
560                self._classifier.add_to_train(X, y)
561
562            if self._classifier._check_ready_for_fit():
563                self._classifier.fit()
564                self.train_complete = True
565
566        return True

616
617        Parameters

619        marker : str

646                self.event_marker_buffer = []
647                self.event_timestamp_buffer = []
648
649                self.marker_count += 1
650                return False  # Stop processing

652        return True  # Continue processing

200    def step(self):
201        """Runs a single BciController processing step.
202
203        See setup() for configuration of processing.
204
205        The method:
206        1. Pulls data from sources (EEG and markers).
207        2. Run a while loop to process markers as long as there are unprocessed markers.
208        3. The while loop processes the markers in the following order:
209            - First checks if the marker is a known command marker from self.marker_methods.
210            - Then checks if it's an event marker (contains commas)
211            - If neither, logs a warning about unknown marker type
212        3. If the marker is a command marker, handles it by calling __handle_command_marker().
213        4. If the marker is an event marker, handles it by calling __handle_event_marker().
214        5. If the command or event marker handling return continue_flag as True, increment the marker count and process the next marker.
215            - Note: If there is an unknown marker type, the marker count is still incremented and processing continues.
216        6. If the command or event marker handling return continue_flag as False, break out of the while loop and end the step.
217
218        Parameters
219        ----------
220        `None`
221
222        Returns
223        ------
224        `None`
225
226        """
227        # read from sources to get new data.
228        # This puts command markers in the marker_data array and
229        # event markers in the event_marker_strings array
230        self._pull_data_from_sources()
231
232        # Process markers while there are unprocessed markers
233        # REMOVE COMMENT: check if there is an available command marker, if not, break and wait for more data
234        while len(self.marker_timestamps) > self.marker_count:
235            # Get the current marker
236            current_step_marker = self.marker_data[self.marker_count]  # String
237            current_timestamp = self.marker_timestamps[self.marker_count]  # Float
238
239            # If messenger is available, send feedback for each marker received
240            if self._messenger is not None:
241                self._messenger.marker_received(current_step_marker)
242
243            # Process markers in order specified in the docstrings
244            # First check if it's a known command marker
245            if current_step_marker in self.marker_methods:
246                continue_flag = self.__handle_command_marker(current_step_marker)
247            # Then check if it's an event marker (contains commas)
248            elif "," in current_step_marker:
249                continue_flag = self.__handle_event_marker(
250                    current_step_marker, current_timestamp
251                )
252            # Otherwise, log a warning about unknown marker type
253            else:
254                # Log warning for unknown marker types
255                logger.warning("Unknown marker type received: %s", current_step_marker)
256
257            # Check if we should continue processing markers in the while loop
258            # if continue_flag is False, then break out of the while loop
259            # else, increment the marker count and process the next marker
260            if continue_flag is False:
261                break
262            else:
263                logger.info("Processed Marker: %s", current_step_marker)
264                self.marker_count += 1

348        # if there isn't a marker source, abort

383        """Pulls eeg samples from source, sanity checks and appends to buffer.
384
385        Parameters

387        `None`
388
389        Returns
390        -------
391        `None`
392
393        """
394
395        # read in the data
396        eeg, timestamps = self.__eeg_source.get_samples()
397        eeg = np.array(eeg)
398        timestamps = np.array(timestamps)
399
400        if eeg.size == 0:
401            return
402
403        if eeg.ndim != 2:
404            logger.warning("discarded invalid eeg data")
405            return
406
407        # if time is in milliseconds, divide by 1000, works for sampling rates above 10Hz
408        try:

410                timestamps = [(timestamps[i] / 1000) for i in range(len(timestamps))]
411
412        # If time units are not defined then define them
413        except Exception:
414            dif_low = -2
415            dif_high = -1
416            while timestamps[dif_high] - timestamps[dif_low] == 0:
417                dif_low -= 1
418                dif_high -= 1
419
420            if timestamps[dif_high] - timestamps[dif_low] > 0.1:
421                timestamps = [(timestamps[i] / 1000) for i in range(len(timestamps))]
422                self.time_units = "milliseconds"
423            else:
424                self.time_units = "seconds"
425
426        # apply time correction, this is essential for headsets like neurosity which have their own clock
427        time_correction = self.__eeg_source.time_correction()
428        timestamps = [timestamps[i] + time_correction for i in range(len(timestamps))]
429
430        self.__data_tank.add_raw_eeg(eeg.T, timestamps)

466    def __process_and_classify(self):
467        """Process the markers and classify the data.
468
469        Parameters
470        ----------
471        `None`
472
473        Returns
474        -------
475        success_string : str
476            String indicating if the processing and classification was successful.
477            Potential values are "Success", "Skip", "Wait".
478            - "Success": The processing and classification was successful.
479            - "Skip": EEG is either absent entirely or contains lost packets.
480            - "Wait": The processing is waiting for more data.
481
482        """
483
484        eeg_start_time, eeg_end_time = self.__paradigm.get_eeg_start_and_end_times(
485            self.event_marker_buffer, self.event_timestamp_buffer
486        )
487
488        # No we actually need to wait until we have all the data for these markers
489        eeg, timestamps = self.__data_tank.get_raw_eeg()
490
491        # Check if there is available EEG data
492        if len(eeg) == 0:
493            logger.warning("No EEG data available")
494            return "Skip"
495
496        # If the last timestamp is less than the end time, then we don't have the necessary EEG to process
497        if timestamps[-1] < eeg_end_time:
498            return "Wait"
499
500        # Check if EEG sampling is continuous over this time period
501        start_indices = np.where(timestamps > eeg_start_time)[0]
502        if len(start_indices) == 0:
503            logger.warning("No timestamps exceed eeg_start_time")
504            return "Skip"
505        start_index = start_indices[0]
506        end_index = np.where(timestamps < eeg_end_time)[0][-1]
507
508        time_diffs = np.diff(timestamps[start_index:end_index])
509        if np.any(time_diffs > 2 / self.fsample):
510            logger.warning("Time gaps in EEG data")
511            return "Skip"
512
513        X, y = self.__paradigm.process_markers(
514            self.event_marker_buffer,
515            self.event_timestamp_buffer,
516            eeg,
517            timestamps,
518            self.fsample,
519        )
520
521        # Add the epochs to the data tank
522        self.__data_tank.add_epochs(X, y)
523
524        # If either there are no labels OR iterative training is on, then make a prediction
525        if self.train_complete:
526            if -1 in y or self.__paradigm.iterative_training:
527                prediction = self._classifier.predict(X)
528                self.__send_prediction(prediction)
529
530        self.event_marker_buffer = []
531        self.event_timestamp_buffer = []
532
533        return "Success"

204
205        The method:
206        1. Pulls data from sources (EEG and markers).
207        2. Run a while loop to process markers as long as there are unprocessed markers.

215            - Note: If there is an unknown marker type, the marker count is still incremented and processing continues.

217

220        `None`

224        `None`
225

230        self._pull_data_from_sources()
