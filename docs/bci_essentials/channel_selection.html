bci_essentials.channel_selection

This module includes functions for selecting channels in order to improve BCI performance.

The EEG data input for each function is a set of windows. The data must be of the shape W x C x S, where:

  • W = number of windows
  • C = number of channels
  • S = number of samples

 17def channel_selection_by_method(
 18    kernel_func,
 19    X,
 20    y,
 21    channel_labels,

 31):
 32    """Passes the BCI kernel function into a wrapper defined by `method`.
 33
 34    Parameters
 35    ----------
 36    kernel_func : function
 37        The classification kernel function which does feature extraction
 38        and classification.
 39        Different functions  are used for MI, P300, SSVEP, etc.
 40    X : numpy.ndarray
 41        Training data for the classifier as windows of EEG data.
 42        3D array containing data with `float` type.
 43

 56    metric : str, *optional*

 69        - Default is `1`.
 70    max_channels : int, *optional*

 94        The predictions from the model.

104
105    """

156def check_stopping_criterion(
157    current_time,
158    nchannels,
159    current_performance_delta,
160    max_time,
161    min_channels,
162    max_channels,
163    performance_delta,
164    print_output=True,
165):
166    """Function to check if a stopping criterion has been met.
167
168    Parameters
169    ----------
170    current_time : float
171        The time elapsed since the start of the channel selection method.
172    nchannels : int
173        The number of channels in the current iteration of the new best channel
174        subset (`len(new_channel_subset)`).
175    current_performance_delta : float
176        The performance delta between the current iteration and the previous.
177    max_time : int
178        The maxiumum amount of time, in seconds, that the function will
179        search for the optimal solution.
180    min_channels : int
181        The minimum number of channels.
182    max_channels : int
183        The maximum number of channels.
184    performance_delta : float
185        The performance delta under which the algorithm is considered to
186        be close enough to optimal.
187    print_output : str, *optional*
188        Flag on whether or not to print output.
189        - Default is `True`.
190
191    Returns
192    -------
193    *bool*
194        Has stopping criterion been met (`True`) or not (`False`).
195
196    """
197    if current_time > max_time:
198        if print_output == "verbose" or print_output == "final":
199            print("Stopping based on time")
200        return True
201
202    elif nchannels <= min_channels:
203        if print_output == "verbose" or print_output == "final":
204            print("Stopping because minimum number of channels reached")
205        return True
206
207    elif nchannels >= max_channels:
208        if print_output == "verbose" or print_output == "final":
209            print("Stopping because maximum number of channels reached")
210        return True
211
212    elif current_performance_delta < performance_delta:
213        if print_output == "verbose" or print_output == "final":
214            print("Stopping because performance improvements are declining")
215        return True
216
217    else:
218        return False

221def sbs(
222    kernel_func,
223    X,
224    y,
225    channel_labels,
226    metric,
227    initial_channels,
228    max_time,
229    min_channels,
230    max_channels,
231    performance_delta,
232    n_jobs,
233    print_output,
234):
235    """The SBS method for channel selection.
236
237    Parameters
238    ----------
239    kernel_func : function
240        The classification kernel function which does feature extraction
241        and classification.
242        Different functions  are used for MI, P300, SSVEP, etc.
243    X : numpy.ndarray
244        Training data for the classifier as windows of EEG data.
245        3D array containing data with `float` type.
246
247        shape = (`W_windows`,`C_channels`,`S_samples`)
248    y : numpy.ndarray
249        Training labels for the classifier.
250        1D array.
251
252        shape = (`nwindows`)
253    channel_labels: list of `str`
254        The set of channel labels corresponding to `C_channels`.
255        A list of strings with length = `C_channels`.
256    metric : str
257        The metric used to measure the "goodness" of the trained classifier.
258    initial_channels : list of `str`
259        Initial guess of channels.
260    max_time : int
261        The maxiumum amount of time, in seconds, that the function will
262        search for the optimal solution.
263    min_channels : int
264        The minimum number of channels.
265    max_channels : int
266        The maximum number of channels.
267    performance_delta : float
268        The performance delta under which the algorithm is considered to
269        be close enough to optimal.
270    n_jobs : int
271        The number of threads to dedicate to this calculation.
272    print_output : str
273        Flag on whether or not to print output. Options are:
274        - `"verbose"`: Print output at each step.
275        - `"final"`: Print output at the end.
276        - `"silent"`: No output.
277
278    Returns
279    -------
280    new_channel_subset : list of `str`
281        The new best channel subset from the list of `channel_labels`.
282    self.clf : classifier
283        The trained classification model.
284    preds : numpy.ndarray
285        The predictions from the model.
286        1D array with the same shape as `y`.
287
288        shape = (`nwindows`)
289    accuracy : float
290        The accuracy of the trained classification model.
291    precision : float
292        The precision of the trained classification model.
293    recall : float
294        The recall of the trained classification model.
295
296
297    """
298    start_time = time.time()
299
300    nwindows, nchannels, nsamples = X.shape
301    sbs_subset = []
302
303    for i, c in enumerate(channel_labels):
304        if c in initial_channels:
305            sbs_subset.append(i)
306
307    previous_performance = 0
308
309    stop_criterion = False
310
311    preds = []
312    accuracy = 0
313    precision = 0
314    recall = 0
315
316    while stop_criterion is False:
317        sets_to_try = []
318        X_to_try = []
319        for c in sbs_subset:
320            set_to_try = sbs_subset.copy()
321            set_to_try.remove(c)
322            sets_to_try.append(set_to_try)
323
324            # get the new X
325            new_X = np.zeros((nwindows, len(set_to_try), nsamples))
326            for i, j in enumerate(set_to_try):
327                new_X[:, i, :] = X[:, j, :]
328
329            # make a list f all subsets of X to try
330            X_to_try.append(new_X)
331
332        # This handles the multiprocessing to check multiple channel combinations at once if n_jobs > 1
333        outputs = Parallel(n_jobs=n_jobs)(
334            delayed(kernel_func)(Xtest, y) for Xtest in X_to_try
335        )
336
337        models = []
338        predictions = []
339        accuracies = []
340        precisions = []
341        recalls = []
342
343        # Extract the outputs
344        for output in outputs:
345            models.append(output[0])
346            predictions.append(output[1])
347            accuracies.append(output[2])
348            precisions.append(output[3])
349            recalls.append(output[4])
350
351        # Get the performance metric
352        if metric == "accuracy":
353            performances = accuracies
354        elif metric == "precision":
355            performances = precisions
356        elif metric == "recall":
357            performances = recalls
358        else:
359            print("performance metric invalid, defaulting to accuracy")
360            performances = accuracies
361
362        # Get the index of the best X tried in this round
363        best_set_index = accuracies.index(np.max(performances))
364
365        # If it starts getting worse
366        # if np.max(accuracies) < best_overall_accuracy:
367        #     stop_criterion = True
368        #     print("accuracy declined, stopping sbs")
369
370        sbs_subset = sets_to_try[best_set_index]
371        new_channel_subset = [channel_labels[c] for c in sbs_subset]
372        model = models[best_set_index]
373        preds = predictions[best_set_index]
374        accuracy = accuracies[best_set_index]
375        # best_overall_accuracy = accuracy
376        precision = precisions[best_set_index]
377        recall = recalls[best_set_index]
378        if print_output == "verbose":
379            print("new subset ", new_channel_subset)
380            print("accuracy ", accuracy)
381            print("accuracies ", accuracies)
382
383        if metric == "accuracy":
384            current_performance = accuracy
385
386        p_delta = current_performance - previous_performance
387        previous_performance = current_performance
388
389        stop_criterion = check_stopping_criterion(
390            time.time() - start_time,
391            len(new_channel_subset),
392            p_delta,
393            max_time,
394            min_channels,
395            max_channels,
396            performance_delta,
397            print_output=True,
398        )
399
400    new_channel_subset = [channel_labels[c] for c in sbs_subset]
401
402    if print_output == "verbose" or print_output == "final":
403        print(new_channel_subset)
404        print(metric, " : ", current_performance)
405        print("Time to optimal subset: ", time.time() - start_time, "s")
406
407    return new_channel_subset, model, preds, accuracy, precision, recall

410def sbfs(
411    kernel_func,
412    X,
413    y,
414    channel_labels,
415    metric,
416    initial_channels,
417    max_time,
418    min_channels,
419    max_channels,
420    performance_delta,
421    n_jobs,
422    print_output,
423):
424    """The SBFS method for channel selection.
425
426    Parameters
427    ----------
428    kernel_func : function
429        The classification kernel function which does feature extraction
430        and classification.
431        Different functions  are used for MI, P300, SSVEP, etc.
432    X : numpy.ndarray
433        Training data for the classifier as windows of EEG data.
434        3D array containing data with `float` type.
435
436        shape = (`W_windows`,`C_channels`,`S_samples`)
437    y : numpy.ndarray
438        Training labels for the classifier.
439        1D array.
440
441        shape = (`nwindows`)
442    channel_labels: list of `str`
443        The set of channel labels corresponding to `C_channels`.
444        A list of strings with length = `C_channels`.
445    metric : str
446        The metric used to measure the "goodness" of the trained classifier.
447    initial_channels : list of `str`
448        Initial guess of channels.
449    max_time : int
450        The maxiumum amount of time, in seconds, that the function will
451        search for the optimal solution.
452    min_channels : int
453        The minimum number of channels.
454    max_channels : int
455        The maximum number of channels.
456    performance_delta : float
457        The performance delta under which the algorithm is considered to
458        be close enough to optimal.
459    n_jobs : int
460        The number of threads to dedicate to this calculation.
461    print_output : str
462        Flag on whether or not to print output. Options are:
463        - `"verbose"`: Print output at each step.
464        - `"final"`: Print output at the end.
465        - `"silent"`: No output.
466
467    Returns
468    -------
469    new_channel_subset : list of `str`
470        The new best channel subset from the list of `channel_labels`.
471    self.clf : classifier
472        The trained classification model.
473    preds : numpy.ndarray
474        The predictions from the model.
475        1D array with the same shape as `y`.
476
477        shape = (`nwindows`)
478    accuracy : float
479        The accuracy of the trained classification model.
480    precision : float
481        The precision of the trained classification model.
482    recall : float
483        The recall of the trained classification model.
484
485
486    """
487    if len(initial_channels) <= min_channels:
488        initial_channels = channel_labels
489
490    start_time = time.time()
491
492    nwindows, nchannels, nsamples = X.shape
493    sbfs_subset = []
494    all_sets_tried = []  # set of all channels that have been tried
495
496    for i, c in enumerate(channel_labels):
497        if c in initial_channels:
498            sbfs_subset.append(i)
499
500    performance_at_nchannels = np.zeros(len(sbfs_subset))
501    best_subset_at_nchannels = [0] * len(sbfs_subset)
502
503    previous_performance = 0
504
505    stop_criterion = False
506
507    preds = []
508    accuracy = 0
509    precision = 0
510    recall = 0
511
512    while stop_criterion is False:
513        # Exclusion Step
514        sets_to_try = []
515        X_to_try = []
516        for c in sbfs_subset:
517            set_to_try = sbfs_subset.copy()
518            set_to_try.remove(c)
519            set_to_try.sort()
520
521            # Only try sets that have not been tried before
522            if set_to_try not in all_sets_tried:
523                sets_to_try.append(set_to_try)
524            else:
525                continue
526
527            # get the new X
528            new_X = np.zeros((nwindows, len(set_to_try), nsamples))
529            for i, j in enumerate(set_to_try):
530                new_X[:, i, :] = X[:, j, :]
531
532            X_to_try.append(new_X)
533
534        # run the kernel function on all cores
535        outputs = Parallel(n_jobs=n_jobs)(
536            delayed(kernel_func)(Xtest, y) for Xtest in X_to_try
537        )
538
539        [all_sets_tried.append(set.sort()) for set in sets_to_try]
540
541        models = []
542        predictions = []
543        accuracies = []
544        precisions = []
545        recalls = []
546
547        # Extract the outputs
548        for output in outputs:
549            models.append(output[0])
550            predictions.append(output[1])
551            accuracies.append(output[2])
552            precisions.append(output[3])
553            recalls.append(output[4])
554
555        # Get the performance metric
556        if metric == "accuracy":
557            performances = accuracies
558        elif metric == "precision":
559            performances = precisions
560        elif metric == "recall":
561            performances = recalls
562        else:
563            print("performance metric invalid, defaulting to accuracy")
564            performances = accuracies
565
566        best_performance = np.max(performances)
567        best_set_index = accuracies.index(best_performance)
568
569        # else:
570        sbfs_subset = sets_to_try[best_set_index]
571        new_channel_subset = [channel_labels[c] for c in sbfs_subset]
572        model = models[best_set_index]
573        preds = predictions[best_set_index]
574        accuracy = accuracies[best_set_index]
575        # best_overall_accuracy = accuracy
576        precision = precisions[best_set_index]
577        recall = recalls[best_set_index]
578
579        performance = performances[best_set_index]
580        if print_output == "verbose":
581            print("Removed a channel")
582            print("new subset ", new_channel_subset)
583            print("accuracy ", accuracy)
584            print("accuracies ", accuracies)
585
586        # If this is the best perfomance at nchannels
587        if performance_at_nchannels[len(sbfs_subset)] < best_performance:
588            performance_at_nchannels[len(sbfs_subset)] = best_performance
589            best_subset_at_nchannels[len(sbfs_subset)] = sbfs_subset
590
591        p_delta = performance - previous_performance
592        previous_performance = performance
593
594        # Conditional Inclusion
595        while True:
596            # Get the length of the set if we were to include an additional channel
597            length_of_resultant_set = len(sbfs_subset) + 1
598            if (
599                length_of_resultant_set > max_channels
600                or length_of_resultant_set == len(channel_labels)
601            ):
602                break
603
604            # Check all of the possible inclusions that do not lead to a previously tested subset
605            potential_channels_to_add = list(range(len(channel_labels)))
606            [potential_channels_to_add.remove(c) for c in sbfs_subset]
607
608            sets_to_try = []
609            X_to_try = []
610
611            for c in potential_channels_to_add:
612                set_to_try = sbfs_subset.copy()
613                set_to_try.append(c)
614                set_to_try.sort()
615
616                if set_to_try not in all_sets_tried:
617                    sets_to_try.append(set_to_try)
618
619                else:
620                    continue
621
622                # get the new X
623                new_X = np.zeros((nwindows, len(set_to_try), nsamples))
624                for i, j in enumerate(set_to_try):
625                    new_X[:, i, :] = X[:, j, :]
626
627                X_to_try.append(new_X)
628
629            # run the kernel on the new sets
630            outputs = Parallel(n_jobs=n_jobs)(
631                delayed(kernel_func)(Xtest, y) for Xtest in X_to_try
632            )
633
634            [all_sets_tried.append(set.sort()) for set in sets_to_try]
635
636            models = []
637            predictions = []
638            accuracies = []
639            precisions = []
640            recalls = []
641            performances = []
642
643            # Extract the outputs
644            for output in outputs:
645                models.append(output[0])
646                predictions.append(output[1])
647                accuracies.append(output[2])
648                precisions.append(output[3])
649                recalls.append(output[4])
650
651            # Get the performance metric
652            if metric == "accuracy":
653                performances = accuracies
654            elif metric == "precision":
655                performances = precisions
656            elif metric == "recall":
657                performances = recalls
658            else:
659                print("performance metric invalid, defaulting to accuracy")
660                performances = accuracies
661
662            best_performance = np.max(performances)
663            best_set_index = accuracies.index(best_performance)
664
665            # if performance is better at that point
666            if performance_at_nchannels[length_of_resultant_set - 1] < best_performance:
667                sbfs_subset = sets_to_try[best_set_index]
668                new_channel_subset = [channel_labels[c] for c in sbfs_subset]
669                model = models[best_set_index]
670                preds = predictions[best_set_index]
671                accuracy = accuracies[best_set_index]
672                precision = precisions[best_set_index]
673                recall = recalls[best_set_index]
674                if print_output == "verbose":
675                    print("Added back a channel")
676                    print("new subset ", new_channel_subset)
677                    print("accuracy ", accuracy)
678                    print("accuracies ", accuracies)
679
680                performance = best_performance
681
682                p_delta = performance - previous_performance
683                previous_performance = performance
684
685                performance_at_nchannels[length_of_resultant_set - 1] = performance
686                best_subset_at_nchannels[length_of_resultant_set - 1] = sbfs_subset
687
688            # if no performance gains, then stop conditional inclusion
689            else:
690                break
691
692            # If any of these are better than the best result for a given n channels then continue
693
694            # Else continue with sbfs subset
695
696        stop_criterion = check_stopping_criterion(
697            time.time() - start_time,
698            len(new_channel_subset),
699            p_delta,
700            max_time,
701            min_channels,
702            max_channels,
703            performance_delta,
704            print_output=True,
705        )
706
707    new_channel_subset = [channel_labels[c] for c in sbfs_subset]
708
709    if print_output == "verbose" or print_output == "final":
710        print(new_channel_subset)
711        print(metric, " : ", performance)
712        print("Time to optimal subset: ", time.time() - start_time, "s")
713
714    return new_channel_subset, model, preds, accuracy, precision, recall

675                if print_output == "verbose":
676                    print("Added back a channel")
677                    print("new subset ", new_channel_subset)
678                    print("accuracy ", accuracy)
679                    print("accuracies ", accuracies)
680
681                performance = best_performance
682
683                p_delta = performance - previous_performance
684                previous_performance = performance
685
686                performance_at_nchannels[length_of_resultant_set - 1] = performance
687                best_subset_at_nchannels[length_of_resultant_set - 1] = sbfs_subset
688
689            # if no performance gains, then stop conditional inclusion
690            else:
691                break
692
693            # If any of these are better than the best result for a given n channels then continue
694
695            # Else continue with sbfs subset
696
697        stop_criterion = check_stopping_criterion(
698            time.time() - start_time,
699            len(new_channel_subset),
700            p_delta,

702            min_channels,
703            max_channels,
704            performance_delta,
705            print_output=True,
706        )

709
710    if print_output == "verbose" or print_output == "final":
711        print(new_channel_subset)
712        print(metric, " : ", performance)
713        print("Time to optimal subset: ", time.time() - start_time, "s")

The SBFS method for channel selection.

Parameters
  • kernel_func (function): The classification kernel function which does feature extraction and classification. Different functions are used for MI, P300, SSVEP, etc.

  • X (numpy.ndarray): Training data for the classifier as windows of EEG data. 3D array containing data with float type.

    shape = (W_windows,C_channels,S_samples)

  • y (numpy.ndarray): Training labels for the classifier. 1D array.

    shape = (nwindows)

  • channel_labels (list of str): The set of channel labels corresponding to C_channels. A list of strings with length = C_channels.
  • metric (str): The metric used to measure the "goodness" of the trained classifier.
  • initial_channels (list of str): Initial guess of channels.
  • max_time (int): The maxiumum amount of time, in seconds, that the function will search for the optimal solution.
  • min_channels (int): The minimum number of channels.
  • max_channels (int): The maximum number of channels.
  • performance_delta (float): The performance delta under which the algorithm is considered to be close enough to optimal.
  • n_jobs (int): The number of threads to dedicate to this calculation.
  • print_output (str): Flag on whether or not to print output. Options are:
    • "verbose": Print output at each step.
    • "final": Print output at the end.
    • "silent": No output.
Returns
  • new_channel_subset (list of str): The new best channel subset from the list of channel_labels.
  • self.clf (classifier): The trained classification model.

  • preds (numpy.ndarray): The predictions from the model. 1D array with the same shape as y.

    shape = (nwindows)

  • accuracy (float): The accuracy of the trained classification model.
  • precision (float): The precision of the trained classification model.
  • recall (float): The recall of the trained classification model.
